About

Description

This is an open source project for “smart parking system”. The purpose of the project is to make cars parking a little bit more smart than a regular parking. The program can monitor all entering and outgoing cars at the parking.

Admin tool

This tool is for controlling the DB. site: https://smartparkingadministration.herokuapp.com/ Purpose: Setting the necessary permissions for the drivers (allow, forbidden) Expiration date // maybe :) Searching for drivers and deleting them from the DB Searching for the cars inside the parking or outside

PC client

This is a program that will be installed on the PC at the entrance to the parking. It’s written in C# language and connects to remote database (DB) to fetch all the cars from the DB. Purpose: Detecting the licence plate from the picture that has been taken at the entrance. Check in the DB the permissions to enter, expiration date etc. Command to the barrier to open
